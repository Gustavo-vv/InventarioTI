# 💻 Sistema de Inventário de TI

Sistema desenvolvido para gerenciamento e controle de equipamentos de TI, permitindo cadastro, rastreamento e controle de usuários.

---

## 🚀 Sobre o projeto

Este projeto foi desenvolvido utilizando C# com Windows Forms e SQL Server, com o objetivo de aplicar na prática conceitos de desenvolvimento de software e banco de dados.

O sistema permite o controle completo de ativos de TI dentro de uma organização, facilitando a organização, rastreamento e gerenciamento dos equipamentos.

---

## 🛠️ Tecnologias utilizadas

- C#
- .NET (Windows Forms)
- SQL Server
- ADO.NET

---

## 💡 Funcionalidades

- 📦 Cadastro de equipamentos
- 🔄 Controle de entrada e saída
- 👤 Sistema de login de usuários
- 🗄️ Integração com banco de dados SQL Server
- 🔍 Consulta de informações

---

## 🗃️ Banco de Dados

O banco de dados foi modelado utilizando conceitos de MER e DER, garantindo organização e integridade das informações.

📌 Para funcionamento correto do sistema:
- Execute o script SQL disponível no projeto
- Crie o banco de dados no SQL Server
